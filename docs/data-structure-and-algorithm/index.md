# 数据结构和算法

在计算机科学中，数据结构（英语：data structure）是计算机中存储、组织数据的方式。

算法（Algorithm）是指解题方案的准确而完整的描述，是一系列解决问题的清晰指令，算法代表着用系统的方法描述解决问题的策略机制。

## 常见的数据结构

- 栈（Stack）
- 队列（Queue）
- 数组（Array）
- 链表（Linked List）
- 树（Tree）
- 图（Graph）
- 堆（Heap）
- 哈希表（Hash table）

## 常见的算法

- 排序算法：快速排序、归并排序、计数排序
- 搜索算法：回溯、递归、剪枝
- 图论：最短路径、最小生成树、网络流建模
- 动态规划：背包问题、最长子序列、计数问题
- 基础技巧：分治、倍增、二分法、贪心算法
